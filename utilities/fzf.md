`FZF` is a fuzzy file finder.

## installation =
### mac os ==
`brew install fzf`
`$(brew --prefix)/opt/fzf/install` for completions

### linux ==
`pacman -S fzf`
add to `.bash_profile` sourcing of these files:
`/usr/share/fzf/key-bindings.bash`
`/usr/share/fzf/completion.bash`
